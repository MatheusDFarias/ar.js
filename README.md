# AR.js Navigation MVP

## Executar

1. Hospede o projeto em HTTPS.
2. Abra no celular.
3. Permita acesso à câmera e localização.
4. Ajuste as coordenadas do destino em js/app.js.

## Tecnologias

- AR.js
- A-Frame
- GPS Location Based AR
